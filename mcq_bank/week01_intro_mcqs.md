# Week 1 MCQs: Introduction to Generative AI

## MCQ 1
Which of the following is the best example of Generative AI?

A. Predicting house price  
B. Classifying email as spam  
C. Generating a product description  
D. Forecasting sales  

Answer: C

Reason:
Generative AI produces new content. Product description generation is a content-generation task.

---

## MCQ 2
Which statement is correct?

A. Predictive AI always generates text  
B. Generative AI only performs classification  
C. Predictive AI predicts labels/values, while Generative AI creates new content  
D. Generative AI cannot be used in business applications  

Answer: C

---

## MCQ 3
What is a major risk of Generative AI systems?

A. They cannot process text  
B. Hallucination  
C. They do not use data  
D. They cannot be deployed  

Answer: B

---

## MCQ 4
What does RAG mainly help with?

A. Increasing randomness  
B. Reducing grounding  
C. Grounding model output in external documents  
D. Removing the need for evaluation  

Answer: C

---

## MCQ 5
Which of the following changes model weights?

A. Prompting  
B. RAG  
C. Fine-tuning  
D. Temperature adjustment  

Answer: C

# Week 1 MCQs

## MCQ 1
Which of the following is an example of Predictive AI?

A. Writing a poem  
B. Generating an image  
C. Predicting whether a transaction is fraudulent  
D. Summarizing a document  

Answer: C

Reason:
Fraud detection predicts a label: fraud or not fraud.

---

## MCQ 2
Which of the following is an example of Generative AI?

A. Forecasting next month’s sales  
B. Classifying an email as spam  
C. Generating a product description  
D. Predicting house price  

Answer: C

Reason:
Generating a product description creates new text content.

---

## MCQ 3
What is the main purpose of RAG?

A. To train the model from scratch  
B. To retrieve relevant external knowledge before generation  
C. To increase model size  
D. To remove the need for prompts  

Answer: B

---

## MCQ 4
Which statement about RAG is correct?

A. RAG always changes model weights  
B. RAG never uses external documents  
C. RAG retrieves relevant information at inference time  
D. RAG is the same as supervised fine-tuning  

Answer: C

---

## MCQ 5
Hallucination in Generative AI means:

A. Model training is complete  
B. The model produces fluent but incorrect or unsupported information  
C. The model refuses to answer  
D. The model retrieves a document  

Answer: B

---

## MCQ 6
Which of the following can reduce hallucination?

A. RAG  
B. Source citation  
C. Better evaluation  
D. All of the above  

Answer: D

---

## MCQ 7
Which of the following changes model weights?

A. Prompting  
B. RAG  
C. Fine-tuning  
D. Retrieval  

Answer: C

---

## MCQ 8
Why is GenAI evaluation harder than traditional classification evaluation?

A. GenAI output can be open-ended and variable  
B. GenAI does not produce any output  
C. Classification has no metrics  
D. GenAI always gives exact answers  

Answer: A

---

## MCQ 9
Which of the following is not usually a GenAI evaluation metric?

A. Faithfulness  
B. Relevance  
C. Safety  
D. RMSE for house price prediction  

Answer: D

---

## MCQ 10
What should a RAG system do when the answer is not present in the retrieved context?

A. Guess confidently  
B. Make up an answer  
C. Say that there is not enough information  
D. Ignore the user query  

Answer: C

---

## MCQ 11
Which is the best description of a complete GenAI pipeline?

A. Prompt → answer  
B. Problem definition → data → model → evaluation → deployment → monitoring  
C. Data → accuracy only  
D. Model training only  

Answer: B

---

Which statement is true?
