# Project Ideas

## Idea 1: Agentic RAG Assistant for Field Service Engineers

### Problem
Field service engineers need quick answers from manuals, SOPs, FAQs, and troubleshooting documents.

### Users
Field service engineers / support engineers.

### Proposed system
User query  
→ intent detection  
→ document retrieval  
→ LLM answer generation  
→ answer validation  
→ final response with source

### Why this is good
- Uses embeddings
- Uses RAG
- Uses agents/tool calling
- Can include evals
- Fits enterprise GenAI use case

### Possible evaluation
- Correctness
- Faithfulness
- Relevance
- Latency
- Safety
